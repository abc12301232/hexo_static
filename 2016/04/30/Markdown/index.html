Syntax

I know there are many sources from the Internet but I still want to recode what I learn today.

Header

Markdown support two syntax of header: Setext and atx.

Setext is like the form of underline, use=(Highest-order) and -(Second-order). For example:

This is an H1
=============

This is an H2
-------------

**Any number of = and - also works.

Atx use 1 to 6 # in the beginning of the sentence, corresponding to H1 to H6. For example,

# This is an H1

## This is an H2

##### This is an H5

You can also add # at the end of the sentence, but the number of # at the end will not affect the Header size:

# This is an H1 #

### This is an H3 ##

##### This is an H5 ###########

Block Quote

Markdown use form of emails’ blockquote, add > at the beginning of every line:

> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
> Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
>
> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
> id sem consectetuer libero luctus adipiscing.

This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

It’s also allow you to add a single > at the beginning of the paragraph:

consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.

Blockquote can have different layer (quote inside a quote), add more >:

> This is the first level of quoting.
>
> > This is nested blockquote.
>
> Back to the first level.

This is the first level of quoting.

This is nested blockquote.

Back to the first level.

You can still use any markdown syntax inside the blockquote:
>

> 1.   First
> 2.   Second
> ---
> Here's some example code:
>
>     return shell_exec("echo $input | $markdown_script");

>

  1. First
  2. Second

Here’s some example code:

return shell_exec("echo $input | $markdown_script");

Many text Editor can build a blockquote easily such as BBEdit.

List

Markdown support Underorder List and Order List.
Underorder List use *, + or - as syntax:

*   Tom
*   Amy
*   Joy

Same as:

-   Tom
-   Amy
-   Joy

And:

+   Tom
+   Amy
+   Joy

Order List use a number follow with a .:

1.   First
2.   Second
3.   Third

Attention that the number you write will not affect the output HTML result, the above list output the following HTMl code:
<ol>
<li>First</li>
<li>Second</li>
<li>Third</li>
</ol>
If you change the list to:

3.   First
2.   Second
1.   Third

Or even:

11.   First
4.   Second
20.   Third

The output format of HTML is completely the same.

If you need to write something like this:

1986. What a great season.

but not a list, you can add a \ between the number and .:

1986\. What a great season.

Code Block

Normally, when we need to insert code inside the content, we will not insert the code directly.
Markdown use <pre> and <code> tags to build a Code block.

In order to build a code block using Markdown, just simply indent with 4 whitespace or 1 tab:

This is a normal paragraph:

This is a code block.

Markdown will turn into:

<p>This is a normal paragraph:</p>

<pre><code>This is a code block.
</code></pre>

A code block will continue until the sentence has no indentation.

Divider

You can add 3 or more *, -, _ to build a divider, the line cannot have other content.
You also can add some whitespace:

***
- - -
_ _ _
*******************
上一篇
下一篇
