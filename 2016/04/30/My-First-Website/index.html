Permission denied

The first problem I face is the most foolish problem and I wasted many time..
After I install git and node.js in the server, I install nvm and npm.
However, when I use npm to install hexo, something wrong:

npm install -g hexo-cli

After I tpye this command, error occured:

sh: 1: node: Permission denied

I don’t know why I login as root and still have this error. Then I try to use the following command:

sudo npm install -g hexo-cli

The following error make me confused:

sudo: npm: command not found

I am not sure that why I can use npm without sudo ONLY…
Fine~ Then I checked whether I have installed Node.js, nvm and npm correctly:

node --version
nvm --version
npm --version

And the result is fine.
Finally, I found a simple way to kill this stuff. link

npm config set user 0
npm config set unsafe-perm true
sudo npm install -g sm

And I can install hexo Eventually although I don’t know whether this command will yield security problem…
